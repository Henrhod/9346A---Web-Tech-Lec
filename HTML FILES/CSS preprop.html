CSS Preprecessors or Framework and polyfills

  • Css Preprecessors
    • Generate css using custom language syntax that includes feature not included in css
      • SASS - allows you to use nested css or cfan apply css withour using brackets
      • SCSS - see https://sass-lang.com/
    • Mixins: information you can reuse
      • eg. mixin abc{ font: italic sans-serif; border: 2px dashed; }
      • call the mixin in s css
      • @include abc(1px, gray)
  • CSS Frameworks
    • provides predefine CSS designs function that can be reused or extend eg. bootstrap, foundation
  • JavaScript Execution Environment
    • Core JavaScript(EcmaScript) language and API
    • (Traditional) Browser Obkect Model(BOM) API(non-standardize)
      • Window, Navigator, Screen, Location and History
    • Document Object Modefl(DOM) API
      • Node, Document, Element, Text, Attr, DocumentType, Comment
      • Event, EventTarget, EventListener
      • CSSStyleSheet, CSSRules
    • Miscellaneous JavaScript Web APIs
      • Geolocation, IndexedDB, Local Storage, Push Notifications, Service Workers, Web Sockets, Web Workers, XMLHttpRequest(AJAX)
